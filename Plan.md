# 装修工人接单管理系统 - 项目详细计划书

## 一、项目整体概述

### 1.1 项目背景

作为装修个体户/装修工人，日常面临以下痛点问题：

- **零散接单**：订单来源分散，缺乏统一管理
- **人工记报价**：手写报价单易出错，难以保存和查询
- **材料成本混乱**：采购记录不清，库存不明，浪费严重
- **利润算不清**：每单赚多少钱靠估算，没有精确核算
- **订单无记录**：历史订单无法追溯，客户信息丢失
- **对账困难**：与客户、供应商对账耗时耗力

本系统基于 **Spring Boot + Vue3** 前后端分离架构，打造专属装修工人业务管理平台，实现**接单管理、材料采购、工程报价、成本核算、利润自动统计**全流程数字化管理，告别纸质记账、口算算账，所有数据自动留存、自动计算。

### 1.2 项目定位

- **目标用户**：个人装修工人、装修工长、小型装修班组（5-20人）
- **架构模式**：Spring Boot 后端 + Vue3 前端 + MySQL 数据库（前后端分离）
- **核心宗旨**：简单易用、自动化计算、适配装修工人使用习惯
- **部署方式**：支持本地部署、云服务器部署，电脑/手机浏览器直接访问
- **数据安全**：本地化部署，数据完全自主掌控

### 1.3 核心功能范围

#### 基础管理
1. 用户登录与权限管理
2. 客户信息管理（增删改查、历史订单关联）
3. 订单接单管理（创建、编辑、状态跟踪）

#### 材料与成本
4. 装修材料库管理（分类、价格维护）
5. 材料采购入库管理
6. 材料库存管理与预警
7. 订单材料消耗绑定与成本核算
8. 人工成本管理（工种、工时、工价）

#### 报价与利润
9. 装修工程智能报价（模板化、自定义）
10. 单项工程/整体工程成本自动核算
11. 每单利润实时计算
12. 月度利润、年度利润统计分析

#### 数据可视化
13. 利润趋势图表（柱状图、折线图）
14. 材料消耗占比分析（饼图）
15. 订单状态分布统计
16. 客户价值分析

#### 高级功能
17. 报价单Excel导出（可发给客户确认）
18. 财务报表Excel导出（月度/年度）
19. 收款记录管理（定金、进度款、尾款）
20. 短信提醒功能（开工提醒、完工提醒）
21. 数据备份与恢复

### 2.3 整体架构流程

```
┌─────────────┐         ┌──────────────┐         ┌─────────────┐         ┌─────────────┐
│   前端页面   │ ─────→  │  Axios请求    │ ─────→  │ Spring Boot │ ─────→  │   MySQL     │
│  (Vue3)     │         │  (HTTP)      │         │   后端      │         │  数据库     │
└─────────────┘         └──────────────┘         └─────────────┘         └─────────────┘
       ↑                        ↑                        ↑                        ↑
       │                        │                        │                        │
       │   JSON数据返回          │   RESTful API          │   MyBatis-Plus        │   SQL查询
       └────────────────────────┴────────────────────────┴────────────────────────┘
```

**请求流程**：
1. 用户在前端页面操作（接单、录材料、算报价）
2. 前端通过 Axios 发起 HTTP 请求，携带 JWT Token
3. Spring Boot 后端接收请求，经过拦截器验证身份
4. Controller 层接收参数并校验，调用 Service 层处理业务逻辑
5. Service 层通过 MyBatis-Plus 操作 MySQL 数据库
6. 后端返回统一格式的 JSON 数据
7. 前端接收数据，更新页面展示、自动计算成本/利润、渲染图表

## 三、数据库设计（核心表结构）
### 3.1 用户表（sys_user）

存储装修工人登录账号信息，支持多用户使用（如工长、班组成员）。

| 字段名 | 数据类型 | 约束 | 说明 |
|--------|---------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 主键ID |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 登录账号（唯一） |
| password | VARCHAR(100) | NOT NULL | 登录密码（BCrypt加密） |
| real_name | VARCHAR(50) | NOT NULL | 真实姓名 |
| phone | VARCHAR(20) | - | 联系电话 |
| role | VARCHAR(20) | DEFAULT 'worker' | 角色（admin/worker） |
| status | TINYINT | DEFAULT 1 | 账号状态（1正常 0禁用） |
| create_time | DATETIME | DEFAULT NOW() | 创建时间 |
| update_time | DATETIME | ON UPDATE NOW() | 更新时间 |

**索引设计**：
- 主键索引：`id`
- 唯一索引：`username`

### 3.2 客户信息表（customer）

存储装修客户基础信息，支持客户画像和历史订单追溯。

| 字段名 | 数据类型 | 约束 | 说明 |
|--------|---------|------|------|
| id | BIGINT | PRIMARY KEY, AUTO_INCREMENT | 主键ID |
| customer_name | VARCHAR(50) | NOT NULL | 客户姓名 |

| address | VARCHAR(200) | - | 装修地址 |
| house_type | VARCHAR(50) | - | 房屋类型（新房/二手房/商铺） |

| remark | TEXT | - | 客户备注（装修需求、偏好等） |

**索引设计**：
- 主键索引：`id`

## 四、核心业务模块详细设计

### 4.1 登录模块

**功能描述**：
- 账号密码登录，支持记住密码
- JWT Token 身份认证，7天有效期
- 简易权限管理（管理员/普通工人）
- 密码BCrypt加密存储，保障安全

**业务流程**：
1. 用户输入账号密码
2. 后端验证账号密码正确性
3. 生成 JWT Token，包含用户ID、角色信息
4. 前端保存 Token 到 localStorage
5. 后续所有接口请求自动携带 Token
6. Token 过期后自动跳转登录页

### 4.2 客户管理模块

**功能清单**：
1. **新增客户**
   - 录入姓名、电话、装修地址
   - 选择房屋类型（新房/二手房/商铺）
   - 填写房屋面积、装修需求备注
   - 记录客户来源（转介绍/网络平台/线下）

2. **客户列表**
   - 支持按姓名、电话模糊搜索
   - 分页展示客户信息
   - 显示客户关联订单数量
   - 支持编辑、删除客户

3. **客户详情**
   - 查看客户完整信息
   - 查看该客户所有历史订单
   - 统计累计消费金额、订单数量

4. **客户画像**
   - 自动标记高价值客户（订单金额大）
   - 记录客户偏好（装修风格、材料喜好）

### 4.3 接单管理模块

**功能清单**：

#### 4.3.1 新建接单
- 选择已有客户或新增客户
- 填写装修类型（家装/工装/局部翻新）
- 详细描述装修项目内容
- 设置开工时间、预计完工时间
- 初始状态：待施工
- 自动生成订单编号（规则：ORD+年月日+序号）

#### 4.3.2 订单列表
- 支持按订单状态筛选（待施工/施工中/已完工/已结算/已取消）
- 支持按时间范围筛选
- 支持按客户姓名、订单号搜索
- 分页展示订单关键信息（订单号、客户、金额、状态）
- 显示订单利润、利润率

#### 4.3.3 订单操作
- **编辑订单**：修改订单基本信息
- **状态变更**：待施工 → 施工中 → 已完工 → 已结算
- **订单作废**：取消无效订单（需填写原因）
- **订单详情**：查看订单完整信息，包括报价、成本、利润明细

#### 4.3.4 订单关联
- 每个订单可绑定多条材料消耗记录
- 每个订单可绑定多个人工成本记录
- 每个订单可生成一份报价单
- 实时计算并显示订单总成本、总利润

### 4.4 材料管理模块

**功能清单**：

#### 4.4.1 材料基础库维护
- 新增、编辑、删除装修常用材料
- 材料分类管理（水电材料/墙面材料/地面材料/木工材料）
- 维护材料规格型号、单位、参考价格
- 设置最低库存预警值
- 标记材料启用/停用状态

#### 4.4.2 材料采购录入
- 选择材料（下拉选择，自动带出单位）
- 录入采购数量、单价、供应商
- 选择付款方式（现金/微信/支付宝/转账）
- 记录发票号码（可选）
- 自动生成采购单号（规则：PUR+年月日+序号）
- **自动更新材料库存和最新采购价**

#### 4.4.3 材料库存查询
- 实时查看各类材料剩余库存
- 库存低于预警值时，首页显示提醒
- 支持按材料名称、分类搜索
- 查看材料采购历史记录

### 4.5 报价管理模块（核心）

#### 4.5.1 报价模板管理

### 4.6 成本核算模块（核心）

系统自动汇总两大成本，实时计算利润：

#### 4.6.1 材料成本
- 订单关联的所有材料消耗金额总和
- 公式：`材料成本 = Σ(材料用量 × 采购单价)`
- 支持查看每种材料的消耗明细

#### 4.6.2 人工成本
- 订单用工天数、单日工价，自动计算总工费
- 公式：`人工成本 = Σ(用工天数 × 单日工价)`
- 支持按工种统计（水电工/泥瓦工/木工/油漆工）

#### 4.6.3 总成本计算
- **总成本 = 材料成本 + 人工成本**
- 实时更新，每次录入成本后自动重新计算

#### 4.6.4 利润计算
- **单笔订单利润 = 报价总金额 - 总成本**
- **利润率 = (利润 ÷ 报价总金额) × 100%**
- 实时显示在订单列表中
- 利润率为负数时，显示红色预警

## 七、核心计算公式（系统自动计算）

### 7.1 材料相关

1. **单项材料采购总价** = 采购数量 × 采购单价
2. **订单材料总成本** = Σ(该订单所有消耗材料数量 × 对应采购单价)
3. **材料库存** = 初始库存 + 采购入库 - 消耗出库

### 7.2 人工相关

4. **单项人工费用** = 用工天数 × 单日工价
5. **订单人工总成本** = Σ(各工种用工天数 × 对应日价)

### 7.3 成本与利润

6. **订单总成本** = 订单材料总成本 + 订单人工总成本
7. **订单利润** = 订单报价总金额 - 订单总成本
8. **订单利润率** = (订单利润 ÷ 订单报价总金额) × 100%
9. **欠款金额** = 订单报价总金额 - 已收金额

### 7.4 统计指标

10. **月度总利润** = Σ(该月所有订单利润)
11. **月度平均利润率** = (月度总利润 ÷ 月度总报价) × 100%
12. **年度总利润** = Σ(该年所有月份利润)
13. **客户累计消费** = Σ(该客户所有订单报价金额)

### 7.5 预警规则

14. **库存预警**：当 `当前库存 < 最低库存` 时，触发预警
15. **利润预警**：当 `利润率 < 10%` 时，显示黄色预警；`利润率 < 0%` 时，显示红色预警
16. **收款预警**：当 `超过预计收款日期 且 未全款` 时，触发催款提醒
